Given a number, return Fizz if the number is a multiple of 3, Buzz if it is a multiple of 5, FizzBuzz if it is a multiple of 3 and of 5, otherwise, return the number given.

Notes:

All numbers are valid Int32, no need to validate them and are positive numbers.


Examples:
Given:
	1
Return:
	1

Given:
	3
Return:
	Fizz

Given:
	5
Return:
	Buzz

Given:
	15
Return:
	FizzBuzz
